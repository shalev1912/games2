פריסה אוטומטית מ־branch: main

איך זה עובד?

כל עדכון לקבצים תחת site/ ב־branch main מפעיל פריסה אוטומטית ל־GitHub Pages. אפשר גם להפעיל ידנית דרך Actions.

עדכון ידני

  1. בצעו שינוי בקבצים תחת site/ ושמרו.
  2. הפעילו סנכרון: scripts/git-sync.bat "הודעת קומיט".
  3. להרצה ידנית של הפריסה: GitHub → Actions → Deploy to GitHub Pages → Run workflow.
