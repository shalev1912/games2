אני רוצה ליצור משחק לימוד רוסית מרובה משתתפים.
בבקשה תבנה לי פרויקט עם:
1. שרת Node.js עם ספריית ws:
   - מאפשר שחקנים להצטרף לחדרים לפי קטגוריה.
   - השרת בוחר מילה רוסית לפי קטגוריה ושולח לשחקנים רמז ומילה מוסתרת.
   - השרת מנהל ניקוד, טיימר, והודעה כשהזמן נגמר או כשמישהו מנחש נכון.
   - שמירה של רשימת שחקנים מחוברים בחדר ועדכון מצב המשחק לכולם.
2. קליינט React:
   - מסך כניסה: כתובת שרת, שם שחקן, בחירת קטגוריה/חדר.
   - מסך Lobby: רשימת שחקנים מחוברים.
   - מסך משחק: מציג מילה מוסתרת, רמז, טיימר, תיבת ניחוש, לוח ניקוד.
   - עיצוב מודרני עם Tailwind: כרטיסיות מעוגלות, צבעים נעימים, רקע גרדיאנט.
3. הסבר איך להריץ מקומית:
   - server → `node server.js`
   - client → `npm install && npm run dev`
